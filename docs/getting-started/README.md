## Prerequisites

Before you begin, make sure you have:

- **Docker** and **Docker Compose** installed
- A terminal (bash, zsh, or similar)
- API keys for at least one LLM provider

## Recommended Path

1. **[Installation](installation.md)** - Build the Docker image
2. **[Quick Start](quick-start.md)** - Set up the `ai` alias and try a command
3. **[Configuration](configuration.md)** - Add your API keys
